Оперативная память (ОЗУ) — это ключевой компонент любой компьютерной системы, обеспечивающий временное хранение данных для быстрого доступа к ним процессора. Скорость работы ОЗУ влияет на производительность устройства, многозадачность и быстродействие программ.

Функции:
- Обеспечение быстрого доступа к данным
- Поддержка работы нескольких программ одновременно
- Уменьшение нагрузки на жесткий диск или SSD
- Ускорение загрузки операционной системы и приложений
Принцип работы:
1. Операционная система загружает данные из накопителя в ОЗУ.
2. Процессор обрабатывает информацию, используя данные из оперативной памяти.
3. После завершения работы данные удаляются из памяти, освобождая место для новых задач.

Такой подход обеспечивает высокую скорость обработки информации по сравнению с работой напрямую с накопителем.

Подсистема памяти — это компоненты, отвечающие за все операции передачи данных между CPU и RAM. Чтобы лучше понимать принцип работы подсистемы памяти, стоит объяснить, из каких компонентов она состоит:
- Контроллер памяти. Элемент, интегрируемый в центральный процессор, который управляет чтением и записью данных.
- Каналы памяти. Логические пути, по которым данные передаются между оперативной памятью и процессором. 
- Модули памяти. Устройства, записывающие и временно хранящие данные для быстрого доступа процессора.
- Топология. Технология последовательного подключения устройств друг к другу. От топологии будет зависеть, в какой модуль RAM будут в первую очередь поступать данные. В серверных системах используется топология Daisy Chain.

Prefetch buffer (буфер предварительной выборки) в памяти DDR — это механизм, позволяющий ядру памяти заранее считывать несколько последовательных битов данных (n-бит) из массива памяти в буфер ввода-вывода за один такт. Этот метод, называемый n-bit prefetch, значительно увеличивает пропускную способность, позволяя передавать данные на высокой частоте шины, в то время как само ядро работает медленнее. 

Основные аспекты Prefetch Buffer:
Принцип работы: DDR считывает данные из ячеек памяти "пакетами" (burst), а не по одному биту за раз, что нивелирует разницу между высокой скоростью шины и относительно медленной скоростью работы массива памяти.
