# **Intel**
## Compute
1. **Hyper-Threading (HT)** — выполнение двух потоков инструкций на одном физическом ядре за счёт дублирования архитектурного состояния. Связана с повышением утилизации вычислительных блоков CPU.
2. **Intel Speed Shift Technology** — аппаратное управление частотами и напряжением процессора без участия ОС. Связана с производительностью и энергоэффективностью.
3. **Transactional Synchronization Extensions (TSX)** — аппаратная поддержка транзакционной памяти для оптимизации синхронизации потоков. Связана с параллельными вычислениями.
4. **Intel Advanced Matrix Extensions (AMX)** — набор матричных инструкций с тайловой архитектурой. Связана с AI/ML и HPC-нагрузками.
5. **Intel Deep Learning Boost (DL Boost)** — SIMD-инструкции для ускорения inference нейросетей. Связана с машинным обучением.
## Memory
1. **Intel Resource Director Technology (RDT)** — мониторинг и управление кэшем и пропускной способностью памяти. Связана с изоляцией и QoS.
2. **Intel Data Streaming Accelerator (DSA)** — аппаратное ускорение копирования и инициализации данных в памяти. Связана с снижением нагрузки на CPU.
3. **Intel In-Memory Analytics Accelerator (IAA)** — ускорение сжатия и аналитических операций над данными в памяти. Связана с big data и in-memory processin
## Aceleration
1. **Intel QuickAssist Technology (QAT)** — аппаратное ускорение криптографии и сжатия даннх. Связана с сетевыми и storage-нагрузками.
2. **Intel Dynamic Load Balancer (DLB)** — аппаратная балансировка очередей задач между ядрами. Связана с высоконагруженными многопоточными системами.
3. **AES-NI (аппаратное ускорение AES)** — набор инструкций для быстрого симметричного шифрования. Связана с высокой пропускной способностью защиты данных.
## Virtualization
1. **Intel VT-x** — аппаратная виртуализация процессора (VMX root / non-root). Связана с запуском виртуальных машин.
2. **Intel VT-x с EPT (Extended Page Tables)** — виртуализация памяти второго уровня. Связана с ускорением работы гипервизоров.
3. **Intel VT-d** — виртуализация ввода-вывода и прямой проброс PCIe-устройств. Связана с IOMMU и изоляцией DMA.
4. **Intel Volume Management Device (VMD)** — логический контроллер управления NVMe через PCIe. Связана с enterprise-хранилищами и hot-plug.

## Security
1. **NX / XD bit (Execute Disable)** — запрет выполнения кода в областях памяти с данными. Связана с защитой от эксплойтов.
2. **Mode-based Execute Control (MBE / MBEC)** — расширение NX для виртуализированных сред. Связана с безопасной виртуализацией.
3. **Intel Software Guard Extensions (SGX)** — изоляция кода и данных в защищённых энклавах. Связана с доверенным выполнением.
4. **Intel Trust Domain Extensions (TDX)** — аппаратная изоляция виртуальных машин от гипервизора. Связана с confidential computing.
5. **Intel Secure Key** — аппаратная генерация криптографических ключей. Связана с безопасностью и криптографией.
6. **Intel Control-flow Enforcement Technology (CET)** — защита потока управления (Shadow Stack, IBT). Связана с предотвращением ROP/JOP-атак.


# AMD
## Compute
- **Simultaneous Multi-Threading (SMT)** — выполнение двух потоков инструкций на одном физическом ядре. Связана с повышением утилизации вычислительных ресурсов CPU.
- **F16C** — инструкции преобразования чисел между FP16 и FP32. Связана с HPC, графикой и ML.
-  **AVX (Advanced Vector Extensions)** — 256-битные SIMD-инструкции. Связаны с высокопроизводительными вычислениями.
-  **AVX2** — расширение AVX с поддержкой целочисленных операций. Связана с HPC, аналитикой и компрессией.
- **AVX-512** — 512-битные SIMD-инструкции (поддержка зависит от поколения CPU). Связаны с AI, HPC и научными нагрузками.

## Memory
- **Secure Memory Encryption (SME)** — аппаратное шифрование содержимого оперативной памяти. Связана с защитой данных от физического доступа.
## Acceleration
- **Galois Field New Instructions (GFNI)** — инструкции для операций в полях Галуа. Связаны с криптографией и кодами коррекции ошибок.
- **Secure Hash Algorithm Extensions (SHA)** — аппаратное ускорение вычисления SHA-хэшей. Связано с криптографической обработкой данных.
- **Аппаратное ускорение AES (AES-NI)** — набор инструкций для быстрого симметричного шифрования AES. Связано с безопасностью и высокой пропускной способностью шифрования.

## Virtualization
-  **AMD Virtualization Technology (AMD-V)** — аппаратная виртуализация процессора (SVM, guest/host mode). Связана с запуском виртуальных машин и гипервизоров.
## **Security**
- **EVP (Enhanced Virus Protection / Execute Disable Bit)** — аппаратный запрет выполнения кода из областей данных. Связана с защитой от эксплойтов.
